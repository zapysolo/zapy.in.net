## Overview

Fefifo is a full-stack agricultural management platform that helps track product inventory across the supply chain.

## Tech Stack

Built with a MERN stack — MongoDB for flexible document storage, Express for the REST API, React on the frontend, and Node.js tying it all together.

## Key Features

- Product inventory CRUD operations
- Supply chain tracking
- Role-based access control
- REST API with proper validation

## Lessons Learned

Working on Fefifo deepened my understanding of schema design for non-relational databases and how to structure a REST API that scales with growing data requirements.
